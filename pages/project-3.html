Open Museum

Concept

Open Museum est une application web permettant d'obtenir les informations concernant les musées de la région PACA. L'application se base sur une architecture REST et a été développée avec la stack MEAN : MongoDB (Atlas), Express.js, Angular 8 et Node.js.

Technologies : MongoDB Express.js Angular Node.js Leaflet Bootstrap
